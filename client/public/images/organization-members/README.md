# Organization member images

Store confirmed member portraits in this folder.

Recommended format:

- File name: the member slug, for example `vivek-khatkar.webp`
- Preferred format: WebP or AVIF
- Minimum size: 800 x 1000 pixels
- Aspect ratio: 4:5 portrait
- Keep each optimized image below approximately 250 KB

Then set the member's `imageUrl` in the matching committee JSON file under
`src/features/public-site/content/organization-members/` to:

`/images/organization-members/vivek-khatkar.webp`

Do not publish sample portraits as real member photographs.
